\# Sistema de Gestión de Inventario Móvil



Aplicación móvil y backend para la gestión de inventario de repuestos, orientada a empresas que requieren control de stock, transferencias internas y reportes automáticos.



\## Funcionalidades

\- Autenticación segura mediante JWT

\- Gestión de stock de repuestos

\- Transferencias de inventario entre usuarios

\- Historial de movimientos

\- Generación de reportes en PDF

\- Envío de notificaciones por correo electrónico

\- Reportes automáticos programados (cron jobs)



\## Arquitectura

\- Aplicación móvil desarrollada en React Native

\- API REST desarrollada en Node.js y Express

\- Base de datos MongoDB con Mongoose

\- Middleware de autenticación y control de acceso



\## Tecnologías

\- React Native (JavaScript)

\- Node.js + Express

\- MongoDB

\- JWT

\- Mongoose



\## Uso

Este proyecto puede adaptarse a distintos contextos empresariales como bodegas, talleres, empresas de mantenimiento o control de activos.



\## Estado del proyecto

Proyecto funcional y en constante mejora.
